1.) Navigate to Assets -> uMMORPG -> Addons -> RCTestingAddones -> SpecializedCrafting ->prefab
2.) Drag theSpecializedCrafting prefab to hierarchy, NOT THE SLOT PREFAB.
3.) Rename to match closely to object what type of item may be crafted here, as to make life easier when working with multiple crafting panels. *not required, just suggested*
4.) Select the SpecializedCrafting in hierarchy, the parent, not the panel.
5.) change size of “Can Be Crafted” to match the amount of items you will be adding.
6.) Add ScriptableRecipes to each element that was just created.
7.) enjoy
